Comet Engine

Minimum Requirements:
-- Windows 7
- Windows Command Prompt
- Windows Powershell (6.2)

Recommended:
-- Windows 10
- Windows Command Prompt
- Windows Powershell (6.2 or later)

If you meet the Recommended Requirements, you will have optimal performance in Comet Engine.
If you only manage to meet the Minimum Requirements, you will have a ok performance.
Windows 2000 is partly compatible with Comet Engine but not all functions may work since it depends on commands that are in Windows 7+.
