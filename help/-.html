Comet Engine

--------------

Parameters


-defaultconfig : Sets Comet Prompt back to default settings at start

This will delete all your settings! Can not be restored.


-highperformance : Runs Comet Prompt at the High Priority

This may slow down other programs on you device.


-login : To be used to login with the loginfeature in Comet Engine

Only to be used in login programs!


-noloading : Skips nearly all the loading stages and goes right to the command input stage.

This means that config files wont be loaded and the files created may be in random locations. (Not all functions will work!)


-notasklist : Stops Comet Prompt from creating a tasklist at startup

Although this may improve performance during startup, if a bug occours, less infomation will be avalible.


-showprelog : Shows the log craeted while startup. Waits for any key to be pressed the continues.


-verfiyfirst : This verfiys Comet Engine before running Comet Prompt.

This is useful if you just created / used a mod or modifed a Comet Engine file. This checks that not any core file were hugely modifed.


These parameters may cause diffrent results compared to noramly running Comet Prompt! This may be in a good or bad way.
